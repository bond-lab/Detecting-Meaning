HG8011: Project 4 (Individual: 15%)

The fourth and final project for HG8011: Detecting Meaning with Sherlock Holmes

This project is divided into two parts: Idioms/MWEs and simplification.

Identify and annotate idioms (and/or) metaphors (10%)
  • The main goal for this task is to help you think about compositionality and how words build up meaning.
  • The secondary goal is to expose you to one more Sherlock Holmes short story --- we hope your new knowledge will make them even more enjoyable.
  • Find your story
  • Read the story.
  • Find 5 examples of multiword expressions where the meaning is non-compositional (i.e. you can't get it directly from the meanings of the individual words)
    e.g. knock up "awaken someone"
    buried in the deepest thought "thinking very deeply"
    NOT guard dog "a dog the guards something"

    • definition/paraphrase
    • comment

Simplification (5%)
  • Rewrite the sentences assigned to you in projects 1,2 and 3 using a restricted vocabulary (defined here to be the most 1,000 common words of English and their inflectional forms).
  • The goal of this task is to test how well you can represent the meaning with a limited vocabulary. So you should not simplify by removing redundant words, just by using a smaller set of words.
  • It is ok to leave capitalized words (April, Sherlock Holmes, ...)
  • It is ok to use either current modern English, or Doyle's Victorian English (long sentences, some inversion). You can split one sentence into many shorter sentences if it makes things flow better.
  • If a sentence (or part of a sentence) is already written only using the restricted vocabulary, you can leave it as is.
  • Everything else must pass the simple writer
    E.g.
    • It is a capital mistake to theorize before you have all the evidence. It biases the judgment.
    • →It is a very bad problem to guess the answer before you have all the facts. It makes your guess not be right.
    • or → It is very bad to guess the answer before you have all the facts. You will guess wrong.

    Simple Writer Explained
  • Try not to use multiword expressions like knock up.
  • Input the text (all the sentences in one go) using the online form.
  • If you decide you want to change things, just input it again, and I will only assess the most recent version.
  • You will be assessed on
    • Completion: does your paraphrase cover all the sentences
    • Fidelity: does it mean the same thing
    • Readability: how well does it flow (probably not too well, it is hard to only use simplified text)
  • Estimated time 1-2 hours.
  • Due date: 2016-04-15 23:59 (Fri)
